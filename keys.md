Commands:

Linux:

1.ls - List the files & directories.

2.pwd - directory path.

3.ls -a - Show the hidden files.

4.cd - current directory.

5.touch - to create an file.

6.mkdir - to create an folder.

7.rm - Delete a file.

8.cp - Copy the file.

9.mv - Move the file.

10.cat - Show the file content.

11.vim - to edit a file.

Permissions
1.chmod - Change permission.

2.chown - Change the file owner.

System info
1.top - shows the running process.

2.htop - Shows the running process using human readable text.

3.df -h - Disk space.

4.free -h - ram usage.

Networking commands
1.ping - checking the network.

2.ifconfig - Show network information.

Process Management
1.ps - show processes.

2.ps aux - All processes.

3.kill PID - Kill the processes.

Bash scripting:

1.#!bin/bash/ - Header.

2.echo - to print a word using (" ").

3.bash.sh - file for bash files.

Conditions
i) If conditions

     Syntax:

             if[];then

                echo " "

             else:

                echo " "

             fi
Loop Statement
i)for loop

     Syntax: 
             for num in 

             do
              
                echo " "
             
             done
ii)While loop

     Syntax:

             Count=
              
             while[];

             do

                 echo " "

             count++

             done

3.crontab -r - Entire cronjob delete.
